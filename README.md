# Team3_6303

На данный момент работает только сервер.
Шаги его запуска описаны ниже:

### Создание .env 
Нужно локально в папке server создать файл .env:

``` console
cd server/
```

В нем нужно прописать:
- Port;
- DB_LINK.

!Важное примечание!
MongoDB работает только через <b>vpn</b>


### Докачать библиотеки
``` console
npm install
```
